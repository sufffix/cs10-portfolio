Simple Slot Machine

Simple slot machine game. Start with 1000 score, press Spin! button to spin. Each spin costs 20 points. Getting 2 of the same symbol awards 40 (net profit of 20), and getting all 3 as the same symbol awards 1000 (net profit of 980).

Score: 1000
